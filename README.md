 ## 项目背景
 
本项目为中国农业大学2019-2020年夏季学期web技术及数据采集与处理实训课程大作业。本项目以<a href="https://you.ctrip.com/">携程旅游景点网站</a>景点数据为爬取目标，抓取景点相关数据，最后通过网页的形式展示爬取结果。
